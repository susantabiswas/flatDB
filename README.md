# flatDb
A simple flat file database like sqlite.


# Dependencies

### make
  #### Windows
    install chocolately and then run:
        ```
        choco install make
        ```
### ruby
  #### Windows
  Install ruby for Windows either using the executable (https://rubyinstaller.org/downloads/) or choco
        ```
        choco install ruby
        ```

        confirm using
        ```
        ruby -v
        ```

### rails
  ```
  gem install rails
  ```

### Install rspec
```
cd <project_dir_root>
```

```
bundle install
```
